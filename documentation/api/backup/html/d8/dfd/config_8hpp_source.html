18 // How to create a new global configuration option:
19 // ================================================
20 // 1) Check if the macro is already defined. The macros should only be changed
21 // in the <project_config.hpp> file
22 // 2) If the macro is not defined, give it a default value here.
23 // 3) Generate a typed constexpr version of the macro using SJ2_DECLARE_CONSTANT
24 // This will check that the desired typed variable, and the macro equal
25 // each other. They wouldn't in cases where the type is bool and the user
26 // uses the value 2 for the macro.
27 // 4) It is recommend, that if there exists an acceptable range for a constant,
28 // then use a static_assert to check that the constant generated from
29 // SJ2_DECLARE_CONSTANT, is within range. For example if kSystemClockRate
30 // can only be between 1Hz and 100Mhz, then kSystemClockRate should be
31 // checked if it is within range.
