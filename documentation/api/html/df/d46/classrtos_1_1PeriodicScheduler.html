bool rtos::PeriodicScheduler::HandlePeriodicSemaphore ( Frequency  frequency,
uint8_t  period 
)
inlineprotected

Handles semaphore of the specified periodic task function to ensure the task does not overrun. Each time the function is invokes, the counter for the respective frequecy in counters_ is incremented.

When the counter matches the period it is reset, at this time, if there is an existing scheduled task and the semaphore is taken before it can be given, then the task has exceeded its allocated run-time; otherwise, the semaphore is given to allow the task to be executed at the specified frequency.

Parameters
frequencyFrequency at which the task executes.
periodSince the FreeRTOS frequency rate is set to 1000Hz, period = 1 would be 1 RTOS tick.
Returns
Returns true if the counters_[frequency] matches the specified period.
