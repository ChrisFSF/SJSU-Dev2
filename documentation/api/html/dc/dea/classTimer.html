TimerIsrCondition  condition,
MatchControlRegister  match_register = kMat0 
)
inlinefinaloverridevirtual

Function that sets a timer of your choice (0-3) to a specific time in milliseconds or micro seconds dependent on initialization Functionality is defined by mode: interrupt, stop, or reset on match.

Parameters
ticksthe count of the timer register (TC) to have an ISR fire
conditionthe condition for which a timer interrupt will occur
match_registerwhich match register (from 0 to 3) should be used for holding the ticks for the condition.
