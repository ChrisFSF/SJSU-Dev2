Detailed Description

template<typename T, const size_t MEMORY_MODEL = etl::memory_model::MEMORY_MODEL_LARGE>
class etl::iqueue_mpmc_mutex< T, MEMORY_MODEL >

This is the base for all queue_mpmc_mutexs that contain a particular type.

Normally a reference to this type will be taken from a derived queue_mpmc_mutex.

etl::queue_mpmc_mutex<int, 10> myQueue;
etl::iqueue_mpmc_mutex<int>& iQueue = myQueue;

This queue supports concurrent access by one producer and one consumer.

TThe type of value that the queue_mpmc_mutex holds.
