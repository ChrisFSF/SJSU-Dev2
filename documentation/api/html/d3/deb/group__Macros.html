Detailed Description

This module is meant for general purpose macros that can be used across the SJSU-Dev2 environment.

#define SJ2_ALIAS (   f)    __attribute__((weak, alias(#f), no_instrument_function))

Similar to the weak attribute, but also gives each function the implementation of the function f.

◆ SJ2_STRINGIFY

#define SJ2_STRINGIFY (   s)    SJ2_STRINGIFY2(s)

These macros are used to stringify define values. For example: 

 #define VALUE true
 #define STRING "value = " STRINGIFY(s) => "value = true"

◆ SJ2_USED

#define SJ2_USED (   variable)    ((void)variable)

SJ2_USED will use void casting as a means to convince the compiler that the variable has been used in the software, to remove compiler warnings about unused variables. NOTE: this will not stop the compiler from optimizing this variable out.

◆ SJ2_WEAK

#define SJ2_WEAK   __attribute__((weak))

Set a function as a "weak" function. This means that if there is another declaration of this exact function somewhere else in the software, the non-weak function will be used instead of the weak function.
